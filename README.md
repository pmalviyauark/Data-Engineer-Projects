# Project 1 -  Academic Insights Using Microsoft Azure ISYS 1123 Performance Survey Analysis
### Overview
- Build a scalable, end-to-end solution using Microsoft Azure to analyse the 16,889 ISYS 1123 student records (2018–2024) for the qualtrics survey and uncover how socioeconomic factors influence academic success.
### Key Objectives – 
- Discover links between household income and academic performance
- Measure COVID-19’s impact on student outcomes
- Deliver interactive dashboards for institutional leaders
- Enable automated, secure, and scalable data processing.
### Azure Services
- Azure Data Factory: Ingest & orchestrate data pipelines
- Azure Data Lake Gen2: Secure, scalable storage
- Azure Databricks: Transform and analyse data (PySpark)
- Azure Synapse Analytics: Build star schema, run queries
- Power BI:  Visualize trends in equity, performance, resilience
